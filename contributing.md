# Contributing to Awesome Loops

This is a curation, not a collection. Every item should be something you can personally recommend.

## What belongs here

- Resources that cover agent loop theory, patterns, or implementation with substance
- Libraries or frameworks that are actively maintained (last commit within 6 months)
- Papers from reputable venues (arXiv, NeurIPS, ICLR, ACL, ICML, etc.)
- Blog posts or guides from practitioners with demonstrated expertise
- Tools with real users and working documentation

## What does not belong here

- Archived or unmaintained repositories
- Promotional content without substance
- Tutorials that duplicate existing links
- Items without public documentation
- Anything you haven't personally evaluated

## Adding an item

1. Fork this repository
2. Add your item in the appropriate section of README.md
3. Use this format: `[Name](url) — One-sentence description of why it's useful.`
4. Make sure the link works and the resource is publicly accessible
5. Open a pull request with a brief explanation of why you're adding it

## Updating an item

If a resource has moved, been archived, or substantially changed, open a PR with the update and a brief note.

## Reporting a broken link

Open an issue with the broken URL and, if you know it, the correct URL or a replacement resource.

## Section organization

If you think a new section is warranted, make the case in your PR. Sections should cover meaningfully distinct territory, not minor variations.
